This website was created as a result of the enthusiasm that Tim Mees had for the type 26 Lotus Elan. Tim tragically died on 24th July 2010. It has been decided to contuniue to offer spares and support for fellow type 26 owners in memory of Tim and it is hoped that you will find this site both interesting and informative.
